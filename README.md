# AI Trading Agent using Reinforcement Learning

This project implements an autonomous stock trading agent using Deep Q Learning.

The agent learns optimal buy, sell, and hold strategies from historical market data.

## Features

- Reinforcement Learning Trading Agent
- Deep Q Network
- Historical Stock Data
- Portfolio Simulation
- Trading Reward Optimization
- Visualization Dashboard

## Tech Stack

Python  
PyTorch  
Reinforcement Learning  
Yahoo Finance API  
Streamlit

## Installation

pip install -r requirements.txt

## Run Project

python src/data_loader.py
python src/train.py

## Run Dashboard

streamlit run app/streamlit_app.py
